# PSC-012Y Quarto Projects

This folder contains a separate, self-contained R/Quarto project for the setup week and for each of the seven instructional units named in the Spring 2026 syllabus.

## Projects

- `00-Setup`
- `01-Unit-1-Foundations-in-R-Quarto-and-Tidy-Data`
- `02-Unit-2-Variable-Types-and-Data-Wrangling`
- `03-Unit-3-Categorical-Data-and-Bar-Charts`
- `04-Unit-4-Continuous-Data-and-Distributions`
- `05-Unit-5-Uncertainty-and-Comparing-Groups`
- `06-Unit-6-Relationships-Between-Continuous-Variables`
- `07-Unit-7-Free-Response-Text-as-Data`

## Shared Structure

Each project includes:

- an `.Rproj` file for opening the folder in RStudio
- a `_quarto.yml` file with project-level Quarto settings
- a starter `.qmd` document tailored to the syllabus topic
- a `README.md` file with quick usage notes
- `data/`, `figures/`, and `notes/` subfolders

The unit projects also include a small local CSV so each project has example data inside its own folder.
