# PSC-012Y Setup

This project aligns with the syllabus setup meeting on April 1, 2026.

## Purpose

Use this folder for installation checks, first-render testing, and documenting whether a local `R` / `RStudio` / `Quarto` workflow or a Posit Cloud workflow is being used.

## Files

- `psc012y-setup.qmd`: starter setup notebook
- `RInstallGuide.qmd`: Quarto recreation of the installation handout PDF
- `Unit_0-setup.qmd`: very short student assignment for learning the basic run/render workflow
- `RInstallGuide.pdf`: original installation guide PDF
- `data/`: optional place for setup screenshots or exported checks
- `figures/`: optional location for rendered images
- `notes/`: optional notes from the setup session
